Detailed Description

@brief Title: Tic-Tac-Toe Game

Description:
The Tic-Tac-Toe Game is a classic two-player strategy game where players take turns marking spaces in a 3x3 grid. The objective is to be the first to complete a row, column, or diagonal with their chosen symbol (typically 'X' or 'O'). The game is known for its simplicity yet strategic depth, making it a popular choice for casual and competitive play.

Features:

User Interface:

The game features a user-friendly graphical interface.
The grid is displayed, and players interact with the game using buttons to make their moves.
Game Mechanics:

Two players take turns making moves.
Each player selects a square to place their symbol ('X' or 'O').
The game alternates between players until a win, draw, or restart condition is met.
Winning Conditions:

A player wins by forming a complete row, column, or diagonal with their symbols.
The game checks for winning conditions after each move.
Restart Functionality:

Players can restart the game at any time to play again.
Credits Display:

The game displays a "Credits" button, providing information about the creators of the game.
How to Play:

Launch the Tic-Tac-Toe Game.
Two players take turns to click on an empty square to place their symbol.
The game checks for a win or draw condition after each move.
If a player wins, the winning cells are highlighted.
Players can restart the game using the "Restart" button.
Information about the creators can be viewed by clicking on the "Credits" button.
Objective:
The main objective of the Tic-Tac-Toe Game is to outmaneuver the opponent and achieve a winning arrangement of symbols. It's a quick and engaging game that tests strategic thinking and pattern recognition.

Target Audience:

Casual gamers looking for a quick and simple game.
Individuals interested in classic strategy games.
Developers interested in studying or modifying the source code.
